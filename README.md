# Eigrutel BD Academy – Index Majeur


Suite d’outils pour organiser, renommer, trier et explorer une documentation visuelle destinée au dessin, à la bande dessinée et à l’illustration.


## Outils

- Eigrutel Index Documentation aka La Morgue

- Eigrutel Photo Renamer

- Eigrutel Documentation Renamer

- Eigrutel Twins


## Usage général

Le flux de travail recommandé est le suivant :


1. éliminer les doublons

2. renommer grossièrement les images

3. affiner le renommage et le classement

4. explorer, rechercher, collecter et étudier la documentation
